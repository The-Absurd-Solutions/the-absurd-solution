# The Absurd Solution

Creative agency portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React Icons

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000

## Build for Production

```bash
npm run build
npm run preview
```
